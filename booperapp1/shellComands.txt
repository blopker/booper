Instructions to run the site locally:

Activate your virtual environment


In your virtual environment terminal type each line individually:
python manage.py makemigrations
python manage.py migrate
python manage.py shell

In the shell (you can copy and paste this whole block):
from booperapp1.models import BoopBoard, Boop
b = BoopBoard(rows=4, columns=4, name="Website BoopBoard")
b.save()
for r in range(4):
    for c in range(4):
        b.boop_set.create(row=r, column=c, booped=False, thisID=((4*r)+c+1))

quit()

In your virtual environment terminal:
python manage.py runserver
